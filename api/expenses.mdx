## Create Expense

<api method="POST" path="/api/expenses" auth="bearer" />

Create a new expense and add it to a group.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

### Request Body

<ParamField body="description" type="string" required>
  Expense description (max 100 characters)
</ParamField>

<ParamField body="amount" type="number" required>
  Expense amount (must be non-negative)
</ParamField>

<ParamField body="group" type="string" required>
  Group ID where this expense belongs
</ParamField>

<ParamField body="splitAmong" type="array" required>
  Array of user IDs to split the expense among
</ParamField>

<ParamField body="category" type="string">
  Expense category. Options: `Food`, `Travel`, `Shopping`, `Entertainment`, `Utilities`, `Other`
  
  Default: `Other`
</ParamField>

### Response

<ResponseField name="_id" type="string">
  Expense's unique identifier
</ResponseField>

<ResponseField name="description" type="string">
  Expense description
</ResponseField>

<ResponseField name="amount" type="number">
  Expense amount
</ResponseField>

<ResponseField name="paidBy" type="string">
  User ID of the person who paid
</ResponseField>

<ResponseField name="splitAmong" type="array">
  Array of user IDs
</ResponseField>

<ResponseField name="group" type="string">
  Group ID
</ResponseField>

<ResponseField name="category" type="string">
  Expense category
</ResponseField>

<ResponseField name="date" type="string">
  ISO 8601 timestamp
</ResponseField>

### Example Request

```bash cURL
curl -X POST https://api.billbuddy.com/api/expenses \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..." \
  -H "Content-Type: application/json" \
  -d '{
    "description": "Groceries for camping",
    "amount": 150.50,
    "group": "507f191e810c19729de860ea",
    "splitAmong": [
      "507f1f77bcf86cd799439011",
      "507f1f77bcf86cd799439012"
    ],
    "category": "Food"
  }'
```

### Example Response

```json 201 Created
{
  "_id": "507f191e810c19729de860eb",
  "description": "Groceries for camping",
  "amount": 150.50,
  "paidBy": "507f1f77bcf86cd799439011",
  "splitAmong": [
    "507f1f77bcf86cd799439011",
    "507f1f77bcf86cd799439012"
  ],
  "group": "507f191e810c19729de860ea",
  "category": "Food",
  "date": "2024-01-15T14:30:00.000Z"
}
```

### Error Responses

```json 500 Internal Server Error
{
  "message": "Server error"
}
```

## Settle Group Expenses

<api method="POST" path="/api/expenses/settle/:groupId" auth="bearer" />

Calculate balances and settle all expenses in a group.

### Headers

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

### Path Parameters

<ParamField path="groupId" type="string" required>
  Group ID to settle
</ParamField>

### Response

<ResponseField name="message" type="string">
  Confirmation message
</ResponseField>

<ResponseField name="balances" type="object">
  Object mapping user IDs to their final balance in the group
</ResponseField>

### Example Request

```bash cURL
curl -X POST https://api.billbuddy.com/api/expenses/settle/507f191e810c19729de860ea \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

### Example Response

```json 200 OK
{
  "message": "Group settled successfully",
  "balances": {
    "507f1f77bcf86cd799439011": 75.25,
    "507f1f77bcf86cd799439012": -75.25
  }
}
```

<Info>
  Positive balance means the user is owed money. Negative balance means the user owes money.
</Info>

### Error Responses

```json 404 Not Found
{
  "message": "Group not found"
}
```

```json 403 Forbidden
{
  "message": "Not authorized to settle this group"
}
```
